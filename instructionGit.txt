1. Создать аккаунт на гитхаб
2. Скачать и установить git
3. Инициализация нового git - git init
4. Добавляете что хотите экспортировать в ваш репозиторий - git add файл(название)
5. Отправляем комментарий который будет ожидать подтвержения - git commit -m ''
6. Соединить с моим проектом - git remote add origin <url>
7. Отправить или подтвердить все - git push origin master
0. Всегда проверять статус git - git status
